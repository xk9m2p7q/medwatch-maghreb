Methodology
MedWatch Maghreb · Project Alhambra
Overview
MedWatch Maghreb is a strategic intelligence monitor tracking political, security, and energy developments across the Western Mediterranean. Items are ingested daily from open sources, scored for priority, and assessed through AI-assisted analysis with human editorial oversight.
Sources & Ingestion
The pipeline queries the GDELT Project API daily, covering global media in Arabic, French, Spanish, and English. Supplementary items are manually curated from key regional outlets including Hespress, Jeune Afrique, Echorouk, Al Jazeera Arabic, Africa Report, RFI Afrique, El Confidencial, and Euractiv. Non-English titles are translated via Groq (Llama 3.3 70B) before storage.
Priority Scoring
Each item receives a priority score based on theme, keyword signals, and source coverage volume:
Security (+3)
Conflict signaling, military moves, sovereignty disputes
Sovereignty (+2)
Energy moves, migration policy, diplomatic recognition
Spain angle (+1)
Direct implications for Spanish or EU foreign policy
Coverage volume (+1–3)
Number of independent sources covering the same event
Country Risk Index
Risk indicators draw from the INFORM Risk Index 2024, a composite humanitarian risk measure by the European Commission's Joint Research Centre. Scale 0–10.

AI Analysis
On-demand analysis is generated via the Anthropic Claude API (claude-sonnet-4-20250514). Each assessment covers four dimensions: Strategic Significance, Key Actors, Watch (30-day horizon), and Spain Angle. Analysis is generated at the user's request.
Editorial Framework
MedWatch operates within a realist IR framework, foregrounding state interests, power dynamics, and structural constraints. Built and curated by Samuel Herize (MA Technology & Global Affairs, IE Madrid / Universidad de Navarra).
